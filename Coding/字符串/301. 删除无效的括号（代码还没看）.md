# [301. 删除无效的括号](https://leetcode.cn/problems/remove-invalid-parentheses/description/)

---

## 题目 (困难)

给你一个由若干括号和字母组成的字符串 s ，删除最小数量的无效括号，使得输入的字符串有效。  

返回所有可能的结果。答案可以按 任意顺序 返回。  

示例 1：  
```
输入：s = "()())()"
输出：["(())()","()()()"]
```

示例 2：  
```
输入：s = "(a)())()"
输出：["(a())()","(a)()()"]
```

示例 3：  
```
输入：s = ")("
输出：[""]
```

提示：  

- 1 <= s.length <= 25
- s 由小写英文字母以及括号 '(' 和 ')' 组成
- s 中至多含 20 个括号

---

## 思路

回溯

---

## 代码

```C++
class Solution {
public:
    vector<string> removeInvalidParentheses(string s) {
        stack<char> stk;
        unordered_set<string> result;
        unordered_set<string> temp;
        vector<string> resultTurn;

        result.insert(s);
    
        int count = 0;
        for (int i = 0; i < s.size(); i++){
            if (s[i] == '('){
                stk.push(s[i]);
            }else if (s[i] == ')'){
                if (!stk.empty()){
                    stk.pop();
                }else{
                    //任意删除i下标即其之前右括号
                    for (string str:result){
                        for (int j = 0; j <= i-count; j++){
                            if (str[j] == ')'){
                                str.erase(str.begin()+j);
                                temp.insert(str);//删除某个后插入temp
                                str.insert(str.begin()+j, ')');//恢复，可以给下一个删不同地方
                            }
                        }
                    }
                    count ++;//result里面字符串变短
                    result = temp;
                    temp.clear();
                }
            }
        }
        //右括号多的情况处理完毕，结果在result里面

        unordered_set<string> result2;//保存全部结果

        if (!stk.empty()){//左括号多了
            for (string s: result){
                stack<char> stk;//局域变量重名
                reverse(s.begin(), s.end());//反向

                unordered_set<string> more;//保存result里某一个string的结果
                more.insert(s);

                int count = 0;
                for (int i = 0; i < s.size(); i++){
                    if (s[i] == ')'){
                        stk.push(s[i]);
                    }else if (s[i] == '('){
                        if (!stk.empty()){
                            stk.pop();
                        }else{
                            //任意删除i下标即其之前左括号
                            for (string str:more){
                                for (int j = 0; j <= i - count; j++){
                                    if (str[j] == '('){
                                        str.erase(str.begin()+j);
                                        temp.insert(str);//删除某个后插入temp
                                        str.insert(str.begin()+j, '(');
                                    }
                                }
                            }
                            count++;//删了 more里面字符串会变短
                            more = temp;
                            temp.clear();
                        }
                    }
                }
                //处理了一个string之后
                for (string str: more){
                    reverse(str.begin(), str.end());
                    result2.insert(str);
                }
            }

            for (const string &last: result2){
                resultTurn.push_back(last);
            }
            return resultTurn;
        }

        for (const string &last: result){
            resultTurn.push_back(last);
        }
        return resultTurn;
    }
};
```

时间复杂度：**O(n)**  
空间复杂度：**O(1)**
